# Contributing to Smartfarm Connect

Thank you for your interest in contributing!

## Ways to Contribute
- Report bugs or feature requests via GitHub Issues
- Fork and submit pull requests for code or documentation
- Help improve tests or add new modules

## Development Workflow
1. Fork and clone the repo
2. Install dependencies: `npm install`
3. Work on your changes in a new branch
4. Run tests: `npm test`
5. Submit your PR and reference any issue it addresses

## Code Style
- Use TypeScript
- Follow naming conventions
- Keep logic modular; prefer services per feature

## Code of Conduct
Please help us keep the community open and welcoming! See CODE_OF_CONDUCT.md.
